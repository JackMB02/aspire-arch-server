📝 Implementation Examples

Required: Include the DraftManager library in your HTML: <script src="/draft-manager.js"></script>

Basic Usage

// Initialize for a form
const draftManager = new DraftManager({
    apiUrl: '/api',                    // Backend API url
    formType: 'contact',               // Form type identifier
    formElement: document.getElementById('contactForm'),
    enableNotifications: true          // Show success/error messages
});

// Save form as draft
draftManager.saveDraft('Optional Draft Name');

// View all drafts
const drafts = await draftManager.getUserDrafts();

// Load a draft
draftManager.loadDraft(draftId);

// Delete a draft
draftManager.deleteDraft(draftId);

Auto-Save Configuration

const draftManager = new DraftManager({
    formType: 'membership',
    formElement: document.getElementById('membershipForm'),
    autosaveInterval: 60  // Auto-save every 60 seconds
});

// Manually start/stop autosave
draftManager.startAutosave();
draftManager.stopAutosave();

Form Submission with Draft Cleanup

document.getElementById('myForm').addEventListener('submit', async (e) => {
    e.preventDefault();
    
    const formData = draftManager.collectFormData();
    
    // Submit to your API
    const response = await fetch('/api/my-form/submit', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(formData)
    });
    
    if (response.ok) {
        // Clear draft after successful submission
        draftManager.clearStoredDraft();
        alert('Form submitted successfully!');
    }
});

Create Draft List UI

async function showMyDrafts() {
    const drafts = await draftManager.getUserDrafts();
    
    const draftList = document.createElement('div');
    draftList.innerHTML = '<h3>Your Saved Drafts:</h3>';
    
    drafts.forEach(draft => {
        const item = document.createElement('div');
        item.innerHTML = `
            <p>${draft.draft_name}</p>
            <button onclick="draftManager.loadDraft(${draft.id})">
                Load
            </button>
            <button onclick="draftManager.deleteDraft(${draft.id})">
                Delete
            </button>
        `;
        draftList.appendChild(item);
    });
    
    document.getElementById('drafts').appendChild(draftList);
}

🔌 Available API Endpoints

Form-Specific Endpoints

POST /api/contact/save-draft
POST /api/get-involved/membership/save-draft
POST /api/get-involved/donate/save-draft
POST /api/get-involved/feedback/save-draft
POST /api/get-involved/ideas/save-draft
POST /api/get-involved/partnership/save-draft

Draft Management Endpoints

POST   /api/drafts/save                          # Save/update draft
GET    /api/drafts/user/:email/:form_type       # Get user drafts by form type
GET    /api/drafts/user/:email                  # Get all user drafts
GET    /api/drafts/:draft_id                    # Get specific draft
DELETE /api/drafts/:draft_id                    # Delete draft
POST   /api/drafts/:draft_id/submit            # Mark draft as submitted
All draft endpoints require email parameter for identification. The DraftManager automatically handles this from either the form email field or localStorage.

✨ Best Practices

1. Always Include Email Field

Drafts are identified by email, so make sure your form has an email field:

<input type="email" name="email" required>

2. Use Meaningful Draft Names

draftManager.saveDraft(`Membership - ${new Date().toLocaleDateString()}`);

3. Enable Auto-Save for Long Forms

const draftManager = new DraftManager({
    formType: 'partnership',
    formElement: document.getElementById('partnershipForm'),
    autosaveInterval: 30  // Save every 30 seconds
});

4. Handle Form Submission Properly

// After successful submission, clear the draft
if (submitResponse.ok) {
    draftManager.clearStoredDraft();
    draftManager.stopAutosave();
}

5. Show Draft Recovery on Page Load

The DraftManager automatically checks for saved drafts and prompts the user to restore them on page load.

6. Test Across Browsers

DraftManager uses localStorage and fetch API. Test on modern browsers (Chrome, Firefox, Safari, Edge).

🔧 Troubleshooting

Drafts Not Saving?

  • ✅ Check email field is not empty
  • ✅ Check browser console for errors
  • ✅ Verify API URL is correctly set
  • ✅ Check CORS settings on backend

Drafts Not Loading on Page Reload?

  • ✅ Check localStorage is not disabled
  • ✅ Check draft was actually saved (check API response)
  • ✅ Verify email matches between sessions
